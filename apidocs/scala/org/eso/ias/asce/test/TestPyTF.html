TestPyTF 

class TestPyTF extends FlatSpec

Test the python tranfer function

The test is done with the TF in TestTransferFunction.py that - sets the output to the same value of the input - sets the mode to the same mode of the input - sets the properties of the output to the properties passed by the ASCE

The test is repeated for each possible IASIO type
