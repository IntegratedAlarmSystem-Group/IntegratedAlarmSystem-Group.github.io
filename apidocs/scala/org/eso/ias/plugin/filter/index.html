  2.  abstract class FilterBase extends Filter

    The basic class for the filters.

    The basic class for the filters. <P> FilterBase keeps a time ordered #history of samples: newest items are in the head of the list. <P> Implementation of filters must take care of removing elements from the history when a new element is added (#sampleAdded(Sample)) and/or when the filtered value is retrieved Filter#apply(). <P> Objects of this class stores also: <UL> <LI>the point in time when the last sample had been submitted (#lastSampleSubmssionTime <LI>the last filtered value returned applying the filter #lastReturnedValue </UL> <P> The filtering is not supposed to perform any I/O but to perform the calculation only on the samples in the history.
