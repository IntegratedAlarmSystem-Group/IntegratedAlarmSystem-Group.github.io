MonitorPointData 

class MonitorPointData extends MonitorPointDataToBuffer

A java POJO representing a monitor point or alarm to be sent to the IAS. <P> A MonitorPointData will be be sent as it to the core of the IAS; it is not meant to be buffered before sending.
