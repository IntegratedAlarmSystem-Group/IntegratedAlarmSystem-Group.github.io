Sample 

class Sample extends AnyRef

A sample of the value read from the remote system. <P> The type of the value of the sample is is Object to allow the developer to send to the core any value without specifying its configuration. The translation to the proper type will be done by the converter that can always access the configuration database. <P> The alarm is a special case: to send alarms to the core the type of #value must be AlarmSample.

A sample is immutable.
