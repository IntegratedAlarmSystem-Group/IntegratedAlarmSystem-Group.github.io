  3.  class SimulatedWeatherStation extends AnyRef

    The SimulatedWeatherStation produces monitor point values for the dumb weather station of the example.

    The SimulatedWeatherStation produces monitor point values for the dumb weather station of the example. <P> This is only an example and as such has nothing to share with a real weather station. <BR> In a real case, the weather station data can be accessible by the API provided by the control system of the weather station like for example a REST API, a dedicated socket, a database, a file. It can also happen that the data must be red directly from a device. <P> For this example we assume that the value changes every SimulatedMonitorPoint#refreshRate mseconds. In a real case the value could be updated any time the user gets the value through the API. In that case, the time to read and update the value must be carefully taken into account in order to offer to the core of the IAS the values in a proper way.
