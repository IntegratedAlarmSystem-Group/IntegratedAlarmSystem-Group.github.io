PublisherTestCommon 

class PublisherTestCommon extends PublisherEventsListener with PublisherEventsListener

Base class for testing the publisher. <P> PublisherTestCommon offers a buffered (#bufferedPublisher) and a unbuffered (#unbufferedPublisher) publisher. <P> A more elegant solution could be that of using only one object implementing the MonitorPointSender interface.
