  •  class DetailedStatsCollector extends AnyRef

    Collects detailed statistics about updated of monitor point values.

    Collects detailed statistics about updated of monitor point values. <P> This objects is notified about every monitor point update received by the plugin from the monitored system.. <BR>When requested, the plugin submits a log with the statistics collected so far and get ready for the next time interval where a time interval is whatever time is between two consecutive calls of #getAndReset(). <P> If there has been no update in the last time interval, no message is logged.
