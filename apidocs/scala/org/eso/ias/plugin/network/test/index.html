    test the UdpPlugin closing the loop from python to the monitor point published by the java plugin: UdpPlugin.py -> UdpPlugin.java -> Plugin.java -> BSDB

    The test setup up the java plugin #udpPlugin, the run the python plugin (MockUdpPlugin.py) that sends some monitor points and alarms.

    The test check if the monitor points sent by MockUdpPlugin.py are finally published by Plugin.java (i.e. received in #dataReceived(MonitorPointData)).
