ReplyKafkaConsumer 

class ReplyKafkaConsumer extends SimpleStringConsumer with KafkaConsumerListener

An helper class to easily consume replies from the kafka reply topic.

The object must be initialized invoking #setUp() before starting to get replies. Once terminated, #tearDown() must be invoked.
