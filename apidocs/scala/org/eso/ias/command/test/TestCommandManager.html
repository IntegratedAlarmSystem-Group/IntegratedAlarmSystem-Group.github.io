org.eso.ias.command.test

TestCommandManager 

class TestCommandManager extends CommandListener with KafkaConsumerListener with AutoCloseable

Test the org.eso.ias.command.CommandManager.

The test sends command to the manager and check - the reception of the reply - that the execution of commands is properly forwarded to the listener of commands

Objects of this class are the listener of the commands so we expect that every time a command is submitted the listener #newCommand(CommandMessage) runs and the reply is sent by the manager This means that objects of this class are senders of commands and replies. But they are alsio the recipt of the commands
