org.eso.ias.sink.ltdb

LtdbKafkaTask 

class LtdbKafkaTask extends SinkTask with Runnable

The task run by the kafka connector for the LTDB.

It saves IASIOs in the Cassandra database.

No need to synchronize because Kafka connectors run on a single thread.

Received IASIOs are saved in the buffer to quickly return from the put method. Asynchronously the thread stores IASIOs in the LTDB. When the flush is executed all the remainings IASIOs in the buffer are saved in the LTDB.
