class AlarmStateTracker extends AnyRef

AlarmStateTracker records the changes of the state of an alarm to be notified to the user by email or some other mechanism.

The update of an alarm is a AlarmState. The state is one of the values defined in Alarm so it records acrtivation, priority, validity and deactivation.

The tracker saves the lstate of the last riund i.e. the actual state before resetting. This allows to know if the state changed since the last reset and also avoid saving states if they did not change from one time inetrval to another

The AlarmStateTracker is immutable. New change of state are added at the head of the list.
