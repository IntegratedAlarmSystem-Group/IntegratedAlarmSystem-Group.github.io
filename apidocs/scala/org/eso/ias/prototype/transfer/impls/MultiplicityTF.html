class MultiplicityTF extends ScalaTransferExecutor[AlarmSample]

Implements the Multiplicity transfer function.

The HIOs in input to this TF are only alarms. The alarm generate by this TF activates when the number of alarms in input is equal or greater then the threshold retrieved from the properties.
