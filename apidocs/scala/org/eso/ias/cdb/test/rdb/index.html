  1.  class TestRdbCdb extends AnyRef

    Test reading and writing data from/to the relational database.

    Test reading and writing data from/to the relational database. <P> Reading and writing is done by CdbReader and CdbWriter implementators as it is done in TestJsonCdb so, in principle the same test can be run for text files and relational database. <BR> The reason to have a separate test is because with hibernate there is no need to explicitly store objects contained in other objects as needed by the CDB on files. Actually, this test should be shorter and easier to read.

    <EM>Note: with the current implementation, running this test will clear the content of the production database
