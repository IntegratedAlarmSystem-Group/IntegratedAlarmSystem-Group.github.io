  5.  class SqlRunner extends AnyRef

    SqlRunner runs a SQL script in the RDB server.

    SqlRunner runs a SQL script in the RDB server. <P> OJDBC drivers runs a single SQL statement so the purpose of this class is to split the statements in a script into a sequence of statements to be executed one at a time. <P> <EM>Note</EM>: The SQL script can contain comments as they are cleaned by objects of this class. The cleaning is "best effort" i.e. not perfect. For example nested comments are not removed.
