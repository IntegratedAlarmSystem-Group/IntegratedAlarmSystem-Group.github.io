org.eso.ias.converter

ValueMapper 

class ValueMapper extends Function[String, String]

The function to convert a string received from a plugin (i.e. a MonitorPointData) to a value to send to the core of the AIS (i.e. a IASValue. <P> If something went wrong during the mapping, the exception is masked and a log issued for investigation. In that case the received monitor point is lost.
